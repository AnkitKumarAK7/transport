# transport


## Endpoints

- `/load` POST: Add a load
- `/load?shipperId=<shipperId>` GET: Retrieve loads by shipperId
- `/load/{loadId}` GET: Retrieve a load by loadId
- `/load/{loadId}` PUT: Update a load by loadId
- `/load/{loadId}` DELETE: Delete a load by loadId
